# 📊 KPI Dashboard: Data Analysis & Visualization

## 📌 Project Overview

This project presents a comprehensive KPI dashboard built to analyze business performance using a retail dataset. It combines data analysis and visualization techniques to uncover insights and present them through an interactive Power BI dashboard.

---

## 🎯 Objective

The goal of this project is to:

* Perform in-depth analysis of business data
* Identify and evaluate key performance indicators (KPIs)
* Understand trends and patterns
* Build an interactive dashboard for data-driven decision-making

---

## 📂 Dataset

* **Dataset Used:** Superstore Sales Dataset (Processed)
* **File:** `data/cleaned_superstore.csv`
* The dataset contains information related to sales, profit, customers, and product categories.

---

## 🧠 Key Performance Indicators (KPIs)

The dashboard includes the following KPIs:

* **Total Sales** – Overall revenue generated
* **Total Profit** – Total profit earned
* **Order Count** – Number of orders placed
* **Average Order Value (AOV)** – Average revenue per order

---

## 📊 Analysis Performed

### 🔹 Sales Trend Analysis

Monthly sales trends were analyzed to identify patterns, growth, and fluctuations over time.

### 🔹 Category-wise Performance

Sales were compared across product categories to identify high-performing segments.

### 🔹 Sales vs Profit Analysis

The relationship between sales and profit was explored to understand profitability trends.

---

## 📈 Dashboard Features

The interactive dashboard includes:

* KPI summary cards for quick insights
* Monthly sales trend visualization (line chart)
* Category-wise sales comparison (bar chart)
* Interactive filters for:

  * Region
  * Category
  * Order Date
